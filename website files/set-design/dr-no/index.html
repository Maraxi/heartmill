  • SUITCASE ROYALE

    music video studio set | personal project

    Inspired by fictional villain 'Dr. No' from the James Bond universe, this studio set is referencing Ken Adam's production design and the 'war room' from the movie 'Dr. Strangelove' by Stanley Kubrik.

    It is following overall sixties/seventies interior design choices and created to be shot in black and white.
